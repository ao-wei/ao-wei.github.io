焦虑探险3-4

再次踏上治愈之路

0 常见误区

0.0 误区一：没有人能理解自己

  1. 可靠的研究统计表明，焦虑症是众多精神障碍中最常见的一种。5%的被调查者一年内有过，7.6%的被调查者生命周期内有过。
  2. 由上述统计数据可见，你并不孤独，也不独特。即使是你认为你身上最独特的症状，也找得出很多跟你相似的朋友。
  3. 由此可见，世界上有众多人能够理解你的感受

0.1 误区二：焦虑症是由童年的创伤等导致（焦虑症的发病原因）

  1. 事实上来说，焦虑症只是神经的敏感化，所有症状都是敏感化的表现而已，这也是你所存在的唯一的问题。
  2. 实际上，焦虑症与其他种种问题无关
  3. 神经敏感化的原因：我们的（不恰当的、被误解的）想法、情绪、行为使得我们的大脑认为我们处在危险之中，使得大脑的恐惧中枢时刻保持在高度警觉的状态，导致人体神经的敏感化，外化表现为我们体会到的各种症状（不舒服的感受）

0.2 误区三：焦虑症会导致危险（生命危险）

  1. 我们错误的认为焦虑症在时时刻刻地摧残和伤害着我们的身体和神经，将我们置于一个有生命危险的状态。
  2. 事实正好相反，焦虑症是一种原始的保护机制。
  3. 人体在面对危险时，有三种非常快速、原始的反应机制。焦虑症中对于战斗和逃跑的体验比较多一些
    1. 战斗（Fight）：肾上腺素分泌
    2. 逃跑（Flight）
    3. 僵住（Freeze）
  4. 焦虑症就是使人体持续地处在了生存模式之中，虽然这种生存模式会让我们感到不舒服，但是其本身是没有任何危险的，它原本就是用于应对危险的
  5. 焦虑症是一种原始的、严厉的爱

0.3 误区四：惊恐发作会导致晕倒

  1. 惊恐发作在最严重的时候也不会让我们晕倒、猝死，它没有实质的危险。
  2. 从医学上来说，人体晕倒是因为大脑供血不足，焦虑症恰恰使得肾上腺素分泌，使我们的血压升高，绝不可能晕倒。同时，肾上腺素在医学上就是用于急救或抢救的药物
  3. 晕倒其实也是人体的一种保护机制，使大脑水平位置降低，增加大脑的供血
  4. “感觉”会晕倒 ≠ 会晕倒

0.4 误区五：焦虑症是治不好的

  1. 焦虑症是可以完全康复的，原因如下
    1. 焦虑症的发作完完全全就是因为人体神经的敏感化，人体神经的敏感化是可以去掉的（去敏感化）
  2. 为什么我们现在的神经还如此敏感？敏感的维持来自于我们对于特定的一些感受、想法、症状的误会和误解，我们一直以来对这些东西有一个错误的阐释和判断。因此焦虑症是一种思维游戏，是一种想法，而这种想法是可以通过我们的练习（改变认知、行为）去改变的
  3. 焦虑症使人体处在生存模式、应激模式之下，身体的耗能增加，但身体其实不愿意持续处于高耗能状态，身体本身也很想恢复正常

0.5 讲在最后

  1. 不要着急，焦虑症不会因为某一句话或某一个道理而康复。我们需要时间和耐心
  2. 哪怕我们有很多症状的不舒服，实际上它是没有任何实质的伤害的，所以当症状出现时，我们就让它来，就像邀请你不喜欢的朋友一样，邀请焦虑症进来，但不用去管他，任凭它做什么，同时带着这些症状（不舒服）去生活，去该做什么做什么

1 新的一年，砍柴、挑水、做饭

1.0 故事梗概

  1. 开悟前，砍柴时想着挑水，挑水时想着做饭，做饭时想着砍柴
  2. 开悟后，砍柴时想着砍柴，挑水时想着挑水，做饭时想着做饭

1.1 和焦虑症之间的关系

  1. 我们现在是小和尚，总是在当下担心着未来的焦虑发作
  2. 焦虑症康复的过程就是开悟的过程，在心态上、想法上、行为上完成从小和尚向老和尚的转变
  3. 我们在吃饭时就想着吃饭，但我们可以带着所有的症状去一起吃饭；我们出门就专注在我们出门要完成的事情之上，但我们可以带着所有的症状去做事；我们睡觉就专注在睡觉之上，我们可以带着焦虑症的症状一起去睡觉
  4. 我们踏着焦虑症的浪（症状），去勇敢的冲浪，任凭浪花的翻涌和颠簸，而不是被浪笼罩
  5. 我们可以睡在恐惧和症状之上

1.2 感悟

  1. 在想法层面上，活在当下，摒弃掉那些无效的、不良的想法。
  2. 在行为层面上，活在当下，摒弃掉那些无效的、不良的行为。
  3. 反复回味、咀嚼那些不好的经历是没有任何帮助的，因为这是把自己放在受害者的位置上，是一种受害者心态。摒弃掉受害者心态，我们是幸存者，是勇敢的人。
  4. 我们不是焦虑症的受害者，我们应该积极主动去改变我们和焦虑症之间的关系，去接受它，而不是把自己视为受害者等待他人向我们伸出援手
  5. 康复过程中的核心心态：对所有的不舒服感到舒服
  6. 过去、未来都是无法着手的，唯有当下我们可以去改变，通过做好当下的每一件事，我们才能够变得更好
