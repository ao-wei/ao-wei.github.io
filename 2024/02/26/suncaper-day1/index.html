Python介绍

Arbitrary Arguments, *args

If you do not know how many arguments that will be passed into your function, add a * before the parameter name in the function definition.

This way the function will receive a tuple of arguments, and can access the items accordingly.

Keyword Arguments

You can also send arguments with the key = value syntax. This way the order of the arguments does not matter.

Arbitrary Keyword Arguments, **kwargs

If you do not know how many keyword arguments that will be passed into your function, add two asterisk: ** before the parameter name in the function definition.

This way the function will receive a dictionary of arguments, and can access the items accordingly.

Combine Positional-Only and Keyword-Only

You can combine the two argument types in the same function.

Any argument before the / , are positional-only, and any argument after the *, are keyword-only.

Python Lambda

A lambda function is a small anonymous function.

A lambda function can take any number of arguments, but can only have one expression.

  1. Syntax

x = lambda arguments : expression

Python PIP

  1. What is PIP?

PIP is a package manager for Python packages, or modules if you like.

  1. What is a Package?

A package contains all the files you need for a module.

Modules are Python code libraries you can include in your project.

Excercise

  1. Unusual List Operation:

Create a single line of code to reverse the elements of a list without using the reverse() method or slicing.

  1. Unique Dictionary Manipulation:

Given a dictionary {‘a’: 1, ‘b’: 2, ‘c’: 3}, create a new dictionary where the keys are the values and the values are the keys.

  1. String Palindrome Check:

Write a function to check if a given string is a palindrome without using any built-in string reversal functions.

  1. Matrix Transposition:

Implement a function to transpose a given matrix without using any built-in functions like zip or numpy.

  1. Custom Object Serialization:

Write a function that takes a Python object and converts it to a string representation in a custom format. Create another function to deserialize the string back into an object.

  1. Recursive Factorial:

Implement a recursive function to calculate the factorial of a given number without using the built-in factorial function.

  1. Advanced List Comprehension:

Given two lists [1, 2, 3] and [4, 5, 6], create a new list containing the product of each pair of elements, but only include the products that are divisible by 3.

  1. Generator Function:
    Create a generator function that yields the Fibonacci sequence up to a given limit without using a list to store the sequence.
  1. Prime Number Generator:

Implement a generator function that yields prime numbers indefinitely without using any external libraries.

  1. Random Password Generator:
    Write a function to generate a random password of a given length without using any external libraries or the secrets module.

  2. Unique Elements in List:
    Write a function to return a new list containing only the unique elements from a given list, without using any built-in functions like set().

  3. List Sorting Without sort():
    Implement a function to sort a list of integers without using the sort() method or the sorted() function.

  4. Dictionary Intersection:
    Given two dictionaries, write a function to find the intersection of their keys and values, creating a new dictionary with common elements.

  5. Pascal’s Triangle:
    Write a function to generate the first n rows of Pascal’s Triangle without using any external libraries.

  6. Anagram Detection:
    Create a function that takes two strings as input and determines if they are anagrams (contain the same characters with the same frequencies) without using any built-in sorting functions.

Linux

Linux文件与目录结构

  1. Linux一切皆文件

linux的文件系统是采用级层式的树状目录结构，在此结构中的最上层是根目录“/”，然后在此目录下再创建其他的目录。

Ubuntu软件安装

apt-get命令适用于 deb 包管理式的 Linux 操作系统（Debian、Ubuntu等），主要用于自动从互联网软件仓库中搜索、下载、安装、升级、卸载软件或操作系统。

  1. 口令登录

Windows可使用CMD进行SSH远程登录，口令登录命令格式为：ssh客户端用户名@服务器ip地址

VIM编辑器

  1. VI、VIM是什么

VI、VIM是Linux系统命令行下的文本编辑器。通过 vi 文件名 或是 vim 文件名 来使用

Linux命令

  1. sudo：以系统管理员的身份执行指令
  2. ls：list的缩写，既可以查看文件夹包含的文件，而且可以查看文件权限（包括目录、文件夹、文件权限）、查看目录信息等。
  3. cd：changeDirectory，切换当前工作目录
  4. pwd：查看当前工作目录路径
  5. mkdir：创建文件夹
  6. rm：删除一个目录中的一个或多个文件或目录
  7. rmdir：从一个目录中删除一个或多个子目录项，不能删除非空目录
